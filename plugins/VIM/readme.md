### 使用方法：

- 安装 Snipmate 插件：[http://www.vim.org/scripts/script.php?script_id=2540]()
- 把 css.snippets 复制到插件的片断目录
- 打开（重起）你的 vim, 打 `sl-rc`，然后按 `tab` 键试试
- 快捷键参考：[https://github.com/sofish/Alice/tree/master/plugins#readme]()
